# COMS-UPLINK

> *UNSC Naval Intelligence · Field Training Module 04*
> **Electromagnetic Spectrum & Radio Systems**

A Halo-themed interactive learning module for 6th grade students covering radio waves, the electromagnetic spectrum, frequency, modulation, and the history of radio communication.

## Modules

| # | Module | Description |
|---|--------|-------------|
| 01 | The Spectrum | Core concepts — frequency, Hertz, and the radio highway |
| 02 | How Radios Work | Carrier waves, modulation (AM vs FM), transmit & receive |
| 03 | Frequency Explorer | Interactive spectrum scanner with live wave visualization |
| 04 | Wave Math | λ = c ÷ f formula with live wavelength & antenna calculator |
| 05 | Build a Signal | Pick a mission type, choose modulation — get evaluated |
| 06 | History | Timeline from Maxwell (1864) to modern wireless |
| 07 | Quiz | 10-question field assessment with progress tracking |

## Usage

Open `index.html` in any modern browser. No dependencies or build step required.

```bash
open index.html
# or serve locally:
python3 -m http.server 8080
```

## Curriculum Alignment

- **Grade:** 6
- **Subject:** Physical Science / Technology
- **Topics:** Electromagnetic spectrum, frequency, wavelength, modulation, wave physics, history of communication technology
